  2. (은행업무 등도 맥에서는 지원이 아직까지는 잘 안되죠) 부트캠프를 이용한 윈도우 설치를 공식적으로 지원하기는 하나 문제는 윈도우 10의 사용자가 아직 많지 않다는 점이 었습니다.
  3. 앞서 소개 드린 바와 같이 윈도우 10은 USB형태로 제공되어져 iso파일의 생성도 쉽지 않았습니다.
  4. 애플 고객센터에서도 아직 윈도우 10에 대한 전문적인 상담이 이루어지지 않아서 더욱 아쉬웠습니다.
  5. 여튼 각설하고 성공 했으니 맥에 윈도우 10을 설치 하는 방법을 공유 해 드리…….
