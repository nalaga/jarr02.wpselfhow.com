서버/하이엔드PC의 주기억을 변혁하는 NVDIMM 기술

  • By jarrsvc
  • on 1월 12, 2018
  • in 미분류

요약

  1. 서버나 PC, 스마트폰 등의 메인 메모리(주 기억)에는 보통 DRAM을 싣는다.
  2. 탑재하는 DRAM의 개수는 용도에 따라 다르다.
  3. 탑재하는 개수가 많고 메인 메모리의 기억 용량이 큰 경우는 수많은 DRAM을 DIMM(Dual-Inline Memory Module)이라고 부르는 모듈을 싣는다.
  4. DIMM의 외형은 직사각형 보드로 마더보드 DIMM 커넥터에 보드를 실어 사용한다.
  5. 기억 용량의 크기가 한정되는 노트PC나 스마트폰 등의 모바일 기기는 DIMM을 싣지 않았다.
