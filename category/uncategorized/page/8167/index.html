윈도우10 CPU 100% 문제 해결법

  • By jarrsvc
  • on 1월 10, 2018
  • in 미분류

요약

  1. 바이오스 업데이트.
  2. 역시나 윈도우 10을 처음 설치했을 때 많이 발생하는 경우로 잘못된 드라이버가 설치되어 있는 경우에도  CPU 100% 현상이 발생할 수 있다.
  3. 서비스 종료 방법은  우선 ‘윈도우 + R’로 실행 창을 띄운 후 ‘services.msc’를 실행.
  4. 만악의 근원(?)인 바이러스, 멀웨어 등등의 악성코드에 감염되어 있다면 CPU 100% 현상이 발생하는 경우도 있다.
  5. 이런 경우는 당연히 백신 프로그램을 통해 치료를 하거나 초기화, 혹은 포맷을 하는 등으로 악성코드 자체를 없애야 한다.
