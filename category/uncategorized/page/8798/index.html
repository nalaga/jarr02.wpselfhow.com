요약

  1. ▶제목: 회로이론-21강의 4단자망 구하는방법4단자망은 입력과 출력이 쌍으로 이루어진 회로라고 생각하시면 됩니다.
  2. 4단자망을 이해하기 위해서는 간단한 행렬 계산방법을 이해하여야 합니다.
  3. 복잡한 행렬공식이 아니기 때문에 아래의 행렬 변환을 간단히 이해하시면 4단자망을 해석하는데 크게 어려움이 없습니다.
  4. 본격적으로 4단자망의 해석을 위해서는 입력을 전압과 전류 (V1,I1)으로 규정하고 4단자정수를 각각 AB C D 로 규정합니다.
  5. 4단자 정수를 통과 하고 나온 출력 전압과 전류를 (V2,I2)로 설정합니다.
