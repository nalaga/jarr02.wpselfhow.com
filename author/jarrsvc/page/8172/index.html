윈도우10 PS/2 키보드 인식 문제 해결 방법

  • By jarrsvc
  • on 1월 10, 2018
  • in 미분류

요약

  1. PS/2 포트는 예전부터 키보드나 마우스를 연결하기 위한 용도로 자주 사용되어왔다.
  2. 하지만 요즘 대부분의 키보드나 마우스는 USB 타입으로 나오는 경우가 많은데 윈도 … 생략 … 못하거나 인식을 했다가 못했다가 하는 등 관련된 문제가 경우가 종종 발생한다.
  3. 따라서 윈도우10의 화상 키보드를 실행해야 하는데, ‘설정 -&gt; 접근성 -&gt; 키보드 탭’으로 이동 하여 화상 키보드를 켜면 사용이 가능하다.
  4. start 값 변경.
  5. 값을 변경했다면 이제 재부팅하여 키보드가 정상적으로 작동하는지 확인해보도록 하자.
