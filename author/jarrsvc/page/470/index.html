요약

  1. 그래서 오늘은 시놀로지 NAS에 사용자를 추가하고 그룹을 설정해서 적절히 권한을 부여하는 … 생략 … 가&gt;  시놀로지 나스 제어판에서 ‘사용자’를 클릭합니다.
  2. 사용자 메뉴에서 ‘생성’ 버튼을 클릭해주시면 됩니다.  사용자 이름과 설명, 이메일, 패스워드를 입력해줍니다.

  4. 이 할당량은 말그대로 해당 폴더에서 사용자가 업로드할 수 있는 파일 크기를 말합니다.
  5. 기본적으로 무제한으로 세팅되어 있지만 1~4테라 이상의 NAS를 사용할 것이므로 할당량은 수백메가에서 수테라바이트까지 설정할 수 있습니다.
