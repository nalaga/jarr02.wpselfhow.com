윈도우 검색(Windows Search)에서 pdf 내용 검색이 안될 때

  • By jarrsvc
  • on 1월 13, 2018
  • in 미분류

요약

  1. 윈도우 검색 (Windows Search)에서 pdf 가 파일 이름으로는 검색이 되는데, 내용으로는 검색이 안될 때가 있습니다.
  2. 이럴 때는 iFilter라는 Windows Search를 위한 plugin에 해당하는 프로그램을 깔아주면 됩니다.
  3. 제어판 &gt; 색인 옵션 &gt; 고급 &gt; 파일 형식 에서 pdf가 어떤 항목으로 되어 있는지 확인하시면 됩니다.
  4. 100% 확실치는 않지만, 제가 보니까 윈도우 10(Windows 10)은 기본적으로 pdf가 내용까지 검색 가능하지만, 윈도우 7 (Windows 7)은 그렇지 않습니다.
  5. (아마 64비트 윈도우 7이 주로 문제인 것 같더라구요) 이런 경우는 PDF iFilter를 따로 설치해주셔야 하고 대표적으로 아래와 같은 프로그램들이 있습니다.
