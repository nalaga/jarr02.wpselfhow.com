유튜브 검색기록 삭제하는 방법

  • By jarrsvc
  • on 1월 10, 2018
  • in 미분류

요약

  1. 유튜브에서 원하는 동영상을 보고 싶을 때 검색을 해서 동영상을 골라 보는데요, 그런데 검색한 키워드들이 기록되어 있어 타인에게 노출될 수 있습니다.
  2. 이번 포스팅에선 유튜브 검색기록 삭제하는 방법을 알아보도록 하겠습니다.유튜브 검색기록 삭제는 유튜브 좌측 상단 메뉴에서 최근 본 동영상을 선택합니다.
  3. 그럼, 유튜브에서 시청 기록, 검색 기록, 댓글 기록을 확인할 수 있는데요, 유튜브 검색 기록 삭제는 검색 기록 탭에서 검색 기록 모두 지우기를 선택하면 됩니다.
  4. 팝업 창이 나타나면 검색 기록 모두 지우기를 선택하면 유튜브 검색기록 삭제가 완료됩니다.
  5. 검색 기록은 한 번 삭제하면 복원이되지 않으니 신중히 생각한 후 삭제하…….

LTE 3G 전환하여 사용하는 방법

  • By jarrsvc
  • on 1월 10, 2018
  • in 미분류

요약

  1. 현대 대부분의 스마트폰은 LTE로 출시되어 빠른 속도의 스마트폰을 사용하고 있지만 LET … 생략 … 이 제한되어 있는 요금제를 사용중이라면 나중을 위해 3G로 전환하여 사용할 수 있습니다.
  2. 이번 포스팅에선 LET 3G 전환 방법을 알아보도록 하겠습니다.LTE 3G 전환은 Data Network Mode에서 가능합니다.
  3. Data Network Mode는 다이얼패드에서 *147359#을 입력하면 진입할 수 있습니다.
  4. Data Network Mode에서 ‘LTE(4G) OFF’를 선택한 후 확인하면 스마트폰이 재부팅됩니다.
  5. 참고로 *147359#은 KT입니다.
